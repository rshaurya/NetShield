# NetShield 🔐

AI-powered URL Threat Detection System

## Features
- Random Forest based ML detection
- Lexical feature extraction
- Hybrid decision engine
- VirusTotal fallback
- Whitelist system

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Train model:
python -m backend.model.train

3. Run analyzer:
python test.py
